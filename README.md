### How to run the programs
1. mount c c:\place_where_8086_assembler_is_stored
2. c:\
3. edit filename.asm
4. type and save your code in asm file and then exit (in file section)
5. tasm filename.asm
6. link filename.obj
7. td filename.exe
8. run>>run
